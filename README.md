# social-network-api

## Description
API for social network application where the user can interact and share their thoughts. Built with express.js for routing, a MongoDB database, and the Mongoose ODM.

## Installation
- Install MongoDB
- Install Mongoose
- Install Express
- Install Node
- Install Nodemon
